# ☕ Brew Haven

A modern and elegant coffee shop landing page built with **React** and **Vite**.

The project focuses on clean design, responsive layout, smooth navigation, and reusable React components. It was created as a portfolio project to practice modern frontend development and UI design.

---

## Live Demo
https://thebrewhaven.vercel.app/


## ✨ Features

- Modern and minimalistic design
- Responsive layout
- Smooth scrolling navigation
- Reusable React components
- Featured coffee section
- About section
- Modern footer with contact information
- Clean and organized project structure

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/SaNbinu/Brew_Haven.git
```

Go to the project folder:

```bash
cd Brew_Haven
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
